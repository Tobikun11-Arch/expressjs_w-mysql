# Contributing to Express.js + MySQL CRUD API

Thank you for considering contributing! 🎉  
This project is open source and welcomes improvements, bug fixes, and new ideas.

---

## 🛠 How to Contribute

1. **Fork the repository**  
   Click the "Fork" button at the top right of this repo to create your own copy.

2. **Clone your fork locally**  
    git clone https://github.com/your-username/repo-link 
    cd project_name

3. **Create a new branch**  
    git checkout -b feature/your-feature-name

4. **Make your changes**  
   Follow the existing code style (ES6 imports, async/await).  
   Keep commits clear and descriptive.  
   Add comments where necessary for clarity.

5. **Test your changes**  
   Run the server with `npm run dev`.  
   Use Postman to test CRUD endpoints.  
   Ensure no breaking changes are introduced.

6. **Commit and push**  
    git add .  
    git commit -m "Add: your feature description"  
    git push origin feature/your-feature-name

7. **Open a Pull Request**  
   Go to your fork on GitHub.  
   Click **Compare & pull request**.  
   Provide a clear description of your changes.

---

## Guidelines

- Keep code clean and readable.
- Use meaningful commit messages.
- Write error handling for database queries.
- Do not commit `.env` or sensitive credentials.
- Respect the MIT License.
